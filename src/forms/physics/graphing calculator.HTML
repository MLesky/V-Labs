Graphing Functions using a Graphing Calculator

Here are the steps to perform the Experiment on Building a Graphing Functions using a Graphing Calculator

    Aim:To graph a function using a graphing calculator and observe its behavior.

    Materials:Graphing calculator Paper and pencil

    Procedures

  1. Turn on the graphing calculator and select the appropriate function mode (e.g. y= mode).
  2. Enter a function into the calculator, such as y = x^2.
  3. Press the "graph" button to graph the function on the calculator screen.
  4. Observe the graph and record any key features, such as the points where the function crosses the x-axis or y-axis, and any maxima or minima.
  5. Use the calculator to explore other functions and observe their behavior.

    6. Observation

    When the function is graphed, you should observe a curve on the calculator screen that represents the behavior of the function. Depending on the function, you may observe key features such as points where the function crosses the x-axis or y-axis, and any maxima or minima.
